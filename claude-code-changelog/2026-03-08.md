# Claude Code 版本更新 - 2026-03-08 检查

## v2.1.71 — 2026-03-07

**新功能：**
- 新增 `/loop` 命令，支持定时循环执行提示或斜杠命令（如 `/loop 5m check the deploy`）
- 新增 cron 调度工具，支持会话内定时提示
- 新增 `voice:pushToTalk` 可重绑定快捷键（默认为空格键），支持 `meta+k` 等组合键
- 扩展 bash 自动批准白名单：`fmt`、`comm`、`cmp`、`numfmt`、`expr`、`test`、`printf`、`getconf`、`seq`、`tsort`、`pr`

**Bug 修复：**
- 修复长时间会话中 stdin 冻结问题
- 修复启用 voice 模式后启动时 5-8 秒卡顿（CoreAudio 初始化阻塞主线程）
- 修复 `/fork` 分支会话共享同一计划文件导致相互覆盖的问题
- 修复图片过大导致上下文溢出的问题
- 修复插件管理、权限提示、Chrome 扩展检测等多个问题

---

## v2.1.70 — 2026-03-06

**新功能：**
- VS Code 新增原生 MCP 服务器管理对话框，在聊天面板中使用 `/mcp` 管理服务器、重连、OAuth 认证
- VS Code 新增计划的完整 Markdown 文档视图，支持添加评论反馈
- 远程控制 `/poll` 频率从 1-2 秒降低到每 10 分钟一次，服务器负载减少约 300 倍

**Bug 修复：**
- 修复使用 `ANTHROPIC_BASE_URL` 配合第三方网关时的 API 400 错误
- 修复自定义 Bedrock 配置文件的 effort 参数错误
- 修复 `ToolSearch` 后模型返回空响应
- 修复 Windows/WSL 下剪贴板损坏非 ASCII 文本（CJK、emoji）的问题
- 修复 Windows 原生二进制文件上 voice 模式加载失败
- 修复 push-to-talk 在会话启动时未激活的问题
