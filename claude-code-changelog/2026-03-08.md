# Claude Code 版本更新 - 2026-03-08

## v2.1.71（2026-03-07）

- 新增 `/loop` 命令，支持定时循环执行提示词或斜杠命令（如 `/loop 5m check the deploy`）
- 新增 cron 定时调度工具，可在会话内设置定时任务
- 新增 `voice:pushToTalk` 按键绑定，可在 `keybindings.json` 中自定义语音激活键（默认空格）
- 扩展 bash 自动审批白名单，新增 `fmt`、`comm`、`cmp`、`numfmt`、`expr`、`test`、`printf`、`getconf`、`seq`、`tsort`、`pr`
- 修复长时间运行会话中 stdin 冻结问题
- 修复语音模式用户启动时 5-8 秒卡顿（CoreAudio 初始化问题）
- 修复 fork 对话共享 plan 文件导致互相覆盖的问题
- 修复 Read 工具中超大图片处理问题
- 修复多个 Claude Code 实例运行时插件安装丢失的问题
- 修复 claude.ai connector OAuth token 刷新时重连问题
- 修复 `/plugin marketplace` 合并冲突和解析错误

## v2.1.70（2026-03-06）

- 修复使用 `ANTHROPIC_BASE_URL` 时第三方网关 API 400 错误
- 修复自定义 Bedrock 推理配置文件的 effort 参数错误
- 修复 `ToolSearch` 后模型返回空响应的问题
- 修复 MCP 服务器的 prompt-cache 失效问题
- 修复 Windows/WSL 上剪贴板损坏非 ASCII 文本（CJK、emoji）的问题
- 修复 Windows 原生二进制上语音模式失败
- 修复旧版 git 上 `/security-review` 命令异常
- 新增 VS Code 中 MCP 服务器管理对话框（使用 `/mcp` 即可管理）
- 大幅降低 Remote Control `/poll` 频率，从 1-2 秒降至每 10 分钟一次（服务器负载降低约 300 倍）
